# Installation

## npm

Tesfy could be installed via [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/). This is the recommended way to get the library.

```bash
npm install tesfy --save
# or
yarn add tesfy
```

## CDN
Tesfy dist files are all avaliables from [jsDelivr](https://www.jsdelivr.com/package/npm/tesfy) or [unpkg](https://unpkg.com/).

```html
<script src="https://cdn.jsdelivr.net/npm/tesfy@1.0.0/dist/tesfy.umd.production.min.js"></script>
<!-- or -->
<script src="https://unpkg.com/tesfy@1.0.0/dist/tesfy.umd.production.min.js"></script>
```

## Initialization
Once imported, instantiate the engine with a `datafile` and optionally a `storage` layer to enable sticky bucketing, a `userId` and a set of `attributes`.

```js
import * as Tesfy from 'tesfy';

const tesfy = new Tesfy.Engine({ datafile, storage, userId, attributes });
```
